{% translate 'Método de Neville' %}

{% translate 'En matemáticas, el algoritmo de Neville es un algoritmo utilizado para la interpolación de polinomios que fue desarrollado por el matemático Eric Harold Neville en 1934. Dados n + 1 puntos, existe un único polinomio de grado ≤ n que pasa por los puntos dados. El algoritmo de Neville evalúa este polinomio.' %}

{% translate 'Calculadora Método de Neville' %}

    {% translate 'Fórmula utilizada' %}

    {% translate 'Fórmula de Neville:' %}

    \[ P_{i,j}(x) = \frac{(x - x_j) \cdot P_{i,j-1}(x) - (x - x_i) \cdot P_{i+1,j}(x)}{x_i - x_j} \]

    {% translate 'Donde:' %}

    • {% translate '\( P_{i,j}(x) \) es el polinomio interpolante basado en los puntos \( x_i \) a \( x_j \)' %}
    • {% translate '\( P_{i,i}(x) = f(x_i) \) son los valores iniciales' %}
    • {% translate 'El resultado de la interpolación es \( P_{0,n}(x) \) para \( n \) puntos' %}

    {% translate "Explicación Método de Neville" %}

    {% translate "1. Preparar los datos:

    " %} {% translate "Supón que se tiene un conjunto de puntos dados (\( x_0, y_0 \)), (\( x_1, y_1 \)), ..., (\( x_n, y_n \)), donde \( x_i \) son los valores de la variable independiente y \( y_i = f(x_i) \) son los valores de la función en dichos puntos.

    " %} {% translate "2. Construcción del polinomio de interpolación de Neville:

    " %} {% translate "El método de Neville es un procedimiento recursivo que construye una tabla de interpolación basada en la siguiente ecuación:
    " %} \[ P_{i,j}(x) = \frac{(x - x_j) P_{i,j-1}(x) - (x - x_i) P_{i+1,j}(x)}{x_i - x_j} \] {% translate "Donde \( P_{i,j}(x) \) representa el polinomio interpolante basado en los puntos \( x_i \) a \( x_j \).

    " %} {% translate "3. Cálculo de los polinomios de interpolación:

    " %} {% translate "La construcción comienza con los valores iniciales:
    " %} \[ P_{i,i}(x) = f(x_i) \] {% translate "Luego, utilizando la ecuación de Neville, se calculan los siguientes niveles de interpolación de manera iterativa:
    " %} {% translate "Para dos puntos \( x_i \) y \( x_{i+1} \), el polinomio de primer grado es:
    " %} \[ P_{i,i+1}(x) = \frac{(x - x_{i+1}) P_{i,i}(x) - (x - x_i) P_{i+1,i+1}(x)}{x_i - x_{i+1}} \] {% translate "Para tres puntos \( x_i, x_{i+1}, x_{i+2} \), el polinomio cuadrático es:
    " %} \[ P_{i,i+2}(x) = \frac{(x - x_{i+2}) P_{i,i+1}(x) - (x - x_i) P_{i+1,i+2}(x)}{x_i - x_{i+2}} \] {% translate "Este proceso se repite hasta construir el polinomio de interpolación de orden \( n \):
    " %} \[ P_{0,n}(x) = \frac{(x - x_n) P_{0,n-1}(x) - (x - x_0) P_{1,n}(x)}{x_0 - x_n} \] {% translate "4. Evaluación del polinomio:

    " %} {% translate "El valor interpolado en \( x \) se obtiene al calcular \( P_{0,n}(x) \), que es la mejor aproximación de la función en dicho punto. Este método es útil porque minimiza el error numérico y evita la necesidad de calcular coeficientes explícitos como en otros métodos de interpolación." %}
